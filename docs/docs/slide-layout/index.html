slide background - solid color

With the bg attribute, you can specify the background color of a slide.

here: bg=#222244

slide background - image

With the bg attribute, you can specify a background image for the slide: after the definition for the solid color, separated by a semicolon.

here: bg=#222244;assets/bgimage.jpg
